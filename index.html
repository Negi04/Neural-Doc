// 02 — capabilities

What Neural Doc Does

Six core capabilities baked into a lean, fast Node.js CLI.

📄
PDF Ingestion

Automatically parses and chunks any PDF document using pdf-parse. Handles complex layouts with clean text extraction.

🔢
Vector Embeddings

Converts text chunks into high-dimensional vectors using Google Generative AI embeddings for semantic understanding.

🌲
Pinecone Vector Store

Stores and retrieves embeddings efficiently with Pinecone — fast semantic search over your entire document.

🤖
Google Gemini LLM

Generates accurate, context-aware answers grounded in your document. Powered by Google's latest Gemini model.

💬
Interactive CLI

Ask questions in a conversational terminal interface. No browser needed — run it anywhere Node.js runs.

⚡
LangChain Pipeline

Clean, modular RAG architecture using LangChain JS. Easy to extend, customize, and integrate into larger systems.
